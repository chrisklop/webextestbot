# WebexTestBot

## Webex Calling API Testing & Development Tool

**WebexTestBot** is a development and testing utility for Webex Calling API integrations. Built for developers working with Webex Calling features like IVR, call handling, and telephony automation.

### Features

- **API Testing Interface** - Test Webex Calling API endpoints
- **IVR Development** - Build and test Interactive Voice Response flows
- **Call Flow Debugging** - Debug call routing and handling logic
- **HTTP Request Testing** - Validate API requests and responses

### Use Cases

- Webex Calling integration development
- IVR menu testing and validation
- API endpoint verification
- Call flow prototyping

### Tech Stack

- HTML/JavaScript
- Webex Calling API integration
- REST API testing
